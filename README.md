# 🎬 Movie Recommender System

This is a Machine Learning Movie Recommendation System built using Python and Streamlit.

## Features
- Recommends similar movies
- Uses cosine similarity
- Interactive web interface
- Movie posters from TMDB API

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- TMDB API

## How to Run

1 Install dependencies

pip install -r requirements.txt

2 Run the app

streamlit run app.py
